# ABSummarizer

ABSummarizer — AI-агент для автоматизации пост-анализа A/B-тестов: генерирует продуктовые выводы и практические рекомендации по результатам экспериментов

## Коротко о проблеме
В рамках работы с постанализами АБ-тестов продуктовые аналитики сталкиваются с общей проблемой - не все продукт менеджеры понимают те выводы, которые аналитики написали на странице wiki с постанализом. Из-за этого продукт менеджеры ходят к аналитикам за пояснениями. Это может занимать до 2-5 рефактоворов wiki страниц с постанализом теста

## Цели проекта
- Автоматически генерировать понятные и точные выводы по A/B-тесту на основании данных эксперимента. Автоматически оформлять wiki страницы с постанализами.
- Генерировать практические продуктовые рекомендации.  
- Уменьшить время, которое аналитик тратит на пояснения.
